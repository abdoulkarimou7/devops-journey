This repository documents my 90-day progression toward building a concrete DevOps profile: automation, CI/CD, infrastructure as code, observability.

## Structure

- `01-bash-linux/` — Bash & Linux fundamentals, scripting
- `02-portfolio-react/` — personal portfolio (React)
- `03-projet-cicd/` — Docker + CI/CD pipeline project
- `04-terraform/` — infrastructure as code (Terraform)
- `05-k8s-monitoring/` — Kubernetes, monitoring, logging

## Usage

Each script is standalone and executable:
\`\`\`bash
chmod +x script.sh
./script.sh [arguments]
\`\`\`
